# Zae Labs

Personal portfolio & brand landing page for **Zae Labs** (Irsya Zaelani) — AI workflows, prompt systems, and web development experiments for modern builders.

## Stack

- **React 18** + **TypeScript**
- **Vite** — build tool
- **Tailwind CSS** — styling
- **lucide-react** — icons

## Features

- Cinematic full-screen hero with cursor-following spotlight reveal (canvas mask)
- Ken Burns zoom animation on hero background
- Smooth entrance animations (blur-fade-up)
- Fixed glassmorphism navigation
- Portfolio hero with typewriter effect
- Experience & certifications with dropdown links
- Searchable tools collection
- Responsive from mobile to desktop
- Premium dark aesthetic throughout

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
